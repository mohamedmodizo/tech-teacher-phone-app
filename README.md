# Tech Teacher Phone App

A phone-friendly web app that researches tech topics from the web and generates:
- notes
- ASCII diagrams
- quizzes
- practice labs
- saved lesson history

## Run
python3 app.py

Then open:
http://127.0.0.1:8000
