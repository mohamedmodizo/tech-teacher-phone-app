# Practice Lab: Data Analysis

## Goal
Create a tiny hands-on exercise to understand Data analysis.

## Task
1. Search for one real product or app that uses Data analysis.
2. Draw its basic flow on paper or in a text file.
3. Build a very small demo related to Data analysis.
4. Write down what worked, what failed, and what you learned.
5. Explain how Data analysis would behave in production.

## Success Check
- You can explain Data analysis clearly.
- You can connect it to a real-world system.
- You can build or sketch a small example.
