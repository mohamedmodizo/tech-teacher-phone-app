# Practice Lab: Docker Networking

## Goal
Create a tiny hands-on exercise to understand docker networking.

## Task
1. Search for one real product or app that uses docker networking.
2. Draw its basic flow on paper or in a text file.
3. Build a very small demo related to docker networking.
4. Write down what worked, what failed, and what you learned.
5. Explain how docker networking would behave in production.

## Success Check
- You can explain docker networking clearly.
- You can connect it to a real-world system.
- You can build or sketch a small example.
